my-forks
========
simplly use (fab update) to sync your github forks repos with source repos

Require
-------
    * fabric : http://fabfile.org/
    * github2 : http://packages.python.org/github2/
    * in ubuntu :

::

    easy_install fabric github2

configuration
-------------
    * open fabfile.py edit my_forks {}

Usage
-----

::

    fab update


thats it !
